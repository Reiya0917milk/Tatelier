## サンプル
### 初期 ～
* Player 1 の演奏スタイルをオート/マニュアルに切り替える
----
/auto 
----
### v0.3.0 ～

* Player n(数値) の演奏スタイルをオート/マニュアルに切り替える
----
/auto 2
----

* 全プレイヤーの演奏スタイルをオート/マニュアルを切り替える
----
/auto all
----

* Player n(数値) の演奏スタイルをオートにする
----
/auto on 3
----

* 全プレイヤーの演奏スタイルをオートにする
----
/auto on all
----
